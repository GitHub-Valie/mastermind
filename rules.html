>_ Rules

... The game is played using:

  • > A decoding console with four digits, and twelve tries
  • > Code digits ranging from 1 to 6

... The player can decide in advance the difficulty, code length and game mode
... The player becomes the codebreaker, and the computer is the codemaker
... A random code of the specified length is generated

... The codebreaker tries to guess the pattern, in both order and digits, within the specified amount of turns
... Each guess is made by submiting an input of digits and pressing the submit button
... Once submitted, the computer will provide feedback on digit right and wrong positioning

... Once feedback is provided, another guess is made; guesses and feedback continue to alternate until either the codebreaker guesses correctly, or trial amount reaches its limit
